I really really dislike statistics… as a Uni subject at least. But I did get the chance the produce something relatively cool in the process of doing an assignment recently.

So what is it? It’s a animation of a “robot” collecting “objects” within a circle. The purpose of the assignment question was to optimise the search pattern to collect the most amount of objects. My algorithm collects about 65% … which isn’t all that fantastic, but good enough for the required standard of the assignment.

The simulation is run using a script I wrote in PHP. The script produces a series of images that I turned into a animation using ImageMagick. If anyone is interested in the code, leave a note in the comment.
