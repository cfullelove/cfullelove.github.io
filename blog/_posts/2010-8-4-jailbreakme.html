There is a new jail-breaking tool in the wild. It’s called JailBreakMe 2.0. In the wake of the legalisation of jail-breaking of iPhones in the US, this tool really makes jail-breaking your iPhone scarily easy - you don’t even need your computer. What is more, it works on the recently released (in Australia anyway) iPhone 4!

JailBreakMe 2.0 works by using a pretty serious security flaw in the Safari web browser that allows arbitrary code execution using PDFs. Meaning, the flaw allows websites that use this exploit to run code they have written on your iPhone.

Jail-breaking is pretty smart use of the security hole, but it definitely opens the door for more sinister applications. For instance, a malicious person could use the same method to gain access to all the information on your iPhone - arbitrary code execution is basically free reign to do anything. And realistically, there is no guarantee that JailBreakMe doesn’t steal information in the process of jail-breaking and installing the required software.

I would urge anyone who is considering using this method to be careful and most definitely BACK UP FIRST.
